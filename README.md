# Дипломный исследовательский веб-сайт

Интерактивное веб-приложение для представления результатов сравнительного исследования машинного перевода (EN→RU) между моделями Gemini, GigaChat и Claude.

## Результаты исследования

- **Победитель**: Claude с итоговым баллом **9.23/10**
- **Второе место**: Gemini — 8.72
- **Третье место**: GigaChat — 6.12

## Структура сайта

1. **Обзор** — общие результаты, графики, профили моделей
2. **Жанры** — анализ по типам текстов (новости, научпоп, официальный, реклама, художественный)
3. **Критерии** — разбор по 5 лингвистическим критериям
4. **Примеры** — интерактивный просмотр 10 текстов с переводами
5. **Методология** — описание методики оценки

## Запуск

```bash
# Переход в директорию проекта
cd /Users/alexesip/Desktop/диплом/tools/diploma_site

# Установка зависимостей (если не установлены)
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## Технологии

- Vite + React 18 + TypeScript
- Tailwind CSS
- Recharts (графики)
- React Router DOM

## Требования

- Node.js 18+
- npm 9+

## Структура данных

Данные хранятся в `src/data/diploma-data.json`:
- `summary` — общая статистика
- `reports` — 10 текстов с оценками
